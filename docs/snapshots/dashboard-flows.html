⚠️ Detección de emergencia
Se evalúa ANTES de cualquier lógica de estado. Palabras clave: urgencia, dolor muy fuerte, no puedo respirar, accidente, desmayo, convulsión...
palabra de emergencia detectada
Respuesta inmediata
Derivación SAMU 131
Mensaje urgente con SAMU 131 + teléfonos del CMC. NO resetea sesión. NO usa Claude.
Palabras que activan emergencia:
emergencia urgencia dolor muy fuerte no puedo respirar estoy grave mucho dolor accidente desmayo convulsión
Tres orígenes posibles
_derivar_humano()
Se activa desde 3 caminos distintos
1. Intent "humano"
Usuario pide hablar con recepción o escribe "4" en el menú
2. Frustración (3 errores)
Escalada automática: error 1 → error 2 → error 3 → recepción
3. Sin disponibilidad
No hay slots en los próximos 60 días para la especialidad pedida
Respuesta
📞 Datos de recepción
Muestra WhatsApp + fijo. Sesión reseteada. Log del contexto original para trazabilidad.
