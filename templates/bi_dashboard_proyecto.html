⚠️ Regla importante
Editar el dashboard
1. Editar dashboard-mensual.html (NO api/static/index.html)
2. Reiniciar contenedor: docker compose restart api
3. El browser cachea agresivamente → usar Cmd+Shift+R o agregar ?nocache=xxx
4. El servidor ya tiene headers no-cache desde 2026-03-27
